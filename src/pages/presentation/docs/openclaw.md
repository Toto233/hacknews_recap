# OpenClaw 详解

## 简介

从 IDE 到个人助理。Claude Code 帮你写代码，OpenClaw 帮你做一切。

## 定位

- 你的个人 AI 助手框架
- 不止写代码，还能操作你的生活
- 24/7 待命，随时响应

## 核心能力

### 消息与沟通

- 多渠道接入：Telegram, Discord, 飞书, WhatsApp, Slack, Signal...
- 主动检查和提醒
- 群聊参与（智能判断何时发言）

### 文件与知识

- 读写本地文件
- 飞书文档操作
- 长期记忆管理
- 知识库管理

### 自动化能力

- 浏览器自动化
- Shell 命令执行
- 节点控制（手机、摄像头、屏幕...）
- 定时任务（heartbeat / cron）

### 集成

- Feishu（飞书）
- 可扩展更多平台

## vibe coding 的终极形态

### 从工具到伙伴

- Cursor/Trae 帮你写代码
- Claude Code 帮你理解项目
- OpenClaw 帮你管理整个数字生活

### 复利效应的极致

- 用得越久，它越懂你
- 代码 + 知识 + 生活，全方位增强
- 机器配合你，而不是你配合机器
