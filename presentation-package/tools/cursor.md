# Cursor 详解

## 简介

Cursor 是一个 AI 原生的 IDE，基于 VS Code 二次开发。

## 核心特点

- 支持多模型：OpenAI、Anthropic、Gemini、xAI、Cursor
- 完整代码库理解能力
- 自主性滑块：从 Tab 补全到完全自主 agent

## 三种使用模式

### 1. Tab 补全
像写代码一样自然，AI 帮你补全代码

### 2. Cmd+K
针对性编辑，指定让 AI 修改某部分代码

### 3. Agent 模式
完全自主，AI 用自己的电脑构建、测试、演示功能

## 实际效果

- NVIDIA 40,000 工程师全面使用
- Salesforce 90% 开发者使用
- Y Combinator 最佳工程师的选择

## 适用场景

- 快速原型开发
- 大型项目开发
- 需要 AI 全程参与的场景
